Несколько вдохновляющих ресурсов

Мегапорталы

  • tympanus.net - сногсшибательные примеры (некоторые приведены здесь ниже). Просто кипяток!
  • freedizain.ru - Потрал клипартов и графики
  • www.designonstop.com - чудный русский ресурс по дизайну
  • Zfort Group - Дайджест интересных материалов из мира веб-разработки и IT за последнюю неделю
  • www.dejurka.ru - "Дежурка", русский ресурс о разработке. Много рекламы, но очень много очень хороших материалов
  • css.yoksel.ru - замечательный русский сайт! Идет по следам западных и иногда улучшает и исправляет ошибки
  • "50 лучших онлайн-курсов и ресурсов для обучения веб-дизайну"
  • 110designsites.pdf - 110 сайтов для дизайнера по версии одной из групп ВК (ссылки в документе идут через ВК! Поэтому, если с ВК что-то не так - придется набирать вручную кадую ссылку в адресной строке)

Демонстрации

  • ZenGarden - демонстрация дизайнерских возможностей CSS (англ.)
  • css-tricks.com/ - крутой ресурс о крутых трюках CSS (англ.)
  • lea.verou.me/css3patterns - декоративные градиентные фоны
  • Очень интересное анимированное меню на чистом CSS
  • Experiments with background-clip: text
  • Еще один вдохновляющий пример крутого анимированного меню

Интересные сайты. Стоит рассмотреть и посмотреть, что внутри

  • www.thrivesolo.com - забавный сайт, на котором картинки - это не картинки, а буквы специального алфавита. И вообще, дизайн интересный
  • clearspending.ru - сколько и на что гос-во тратит бюджетные деньги

Онлайн-сервисы для создания интересных элементов и эффектов

  • devdocs.io - английский справочник обо всем на свете, что связано с разработкой
  • psd2htmlconverter.com/ru - автоматический конвертер из PSD в HTML/CSS
  • www.csstextwrap.com - выравниваем текст очень замысловатым образом (англ.)
  • wowslider.com/ru - прекрасный русский ресурс о слайдерах
  • ie.microsoft.com/testdrive/Graphics/hands-on-css3 - говорите, Microsoft - это плохо? - а вы зайдите, посмотрите!
  • livetools.uiparade.com/button-builder - сервис для создания красивых кнопок
  • www.myfonts.com/WhatTheFont - сервис для определения шрифта по рисунку (аналоги ищите по запросу WhatFont)
  • csswarp.eleqtriq.com - сервис для создания искривленных надписей на CSS (внимание! кода будет ОЧЕНЬ много!)
