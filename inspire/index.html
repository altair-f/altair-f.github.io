Inspiration Sources

Sublime Text

Перспективный редактор, современный эталон редакторов. Особенно популярен среди веб-разработчиков. Имеется бесплатная версия.

GitHub

Ресурс планетарного масштаба для хранения и обмена фрагментами кода. Основан на Git. Регистрация и открытое хранение - бесплатны.

MarkDown

Облегченный язык разметки, конвертируемый в HTML. Вы писатель, журналист, аналитик? Вам надо много писать в сети, но не хочется писать код html? MarkDowm для вас! Код легко набирать и очень легко читать!
